## Getting Started

### Prerequisites

- Node.js 18+
- npm atau yarn

### Installation

1. Clone repository:
```bash
git clone https://github.com/muhamad-surayuda/K4-Partners.git
cd K4-Partners
```

2. Navigate ke frontend:
```bash
cd app/frontend
```

3. Install dependencies:
```bash
npm install
```

### Development

Jalankan development server:
```bash
npm run dev
```

Website akan berjalan di `http://localhost:3000`

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```
